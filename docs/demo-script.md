# PRISM Demo Script

Use this when recording a video or live demo for recruiters.

## 60-second demo (tight)
1. Landing page (5s)
   - Say: "PRISM is AI decision intelligence. It analyzes ideas across five lenses and reveals blind spots."
2. Analyze form (15s)
   - Paste a prepared prompt from prompt-examples.md.
   - Say: "The model streams results in real time and returns a structured verdict."
3. Streaming results (15s)
   - Show the loading state and first lens cards filling.
   - Say: "Each lens includes a score, headline, and blind spot."
4. Final results (15s)
   - Point to composite score and verdict.
   - Say: "PRISM is built for clarity, not encouragement."
5. Dashboard history (10s)
   - Show past analyses and the average score.

## 3-5 minute recruiter walkthrough
1. Positioning (20s)
   - "PRISM evaluates decisions across five structured lenses: viability, risk, timing, differentiation, and personal fit."
2. Create an analysis (60s)
   - Fill the form with a real idea.
   - Explain how context improves output quality.
3. Streamed output (60s)
   - Call out server-sent events and structured JSON validation.
4. Lens breakdown (60s)
   - Explain how blind spots turn into action items.
5. History and product completeness (40s)
   - Show the dashboard and repeatability of results.
6. Close (20s)
   - "This is a production-grade full-stack AI product with auth, rate limits, and persistent history."

## Live demo checklist
- Use a dedicated demo account
- Have at least 3 complete analyses in history
- Keep one recent analysis for a fresh run
- Ensure GROQ_API_KEY is set
- Use a clean browser profile
